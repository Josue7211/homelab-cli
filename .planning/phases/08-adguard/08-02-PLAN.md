**cmd_unblock()** (per D-06, AG-06):
- Usage: `adguard unblock <domain>`
- Validate 1 arg or die: `die "Usage: adguard unblock <domain>"`
- The block rule format is `||domain^` -- need to find and remove it from user_rules
- Get current rules: `ag_api GET /filtering/status` and extract user_rules via python3
- Check if `||{domain}^` exists in the rules list. If not, die "No block rule found for: $domain"
- Call `confirm_action "Remove block rule for '$domain'?"`
- Build new rules list with the matching rule removed, using python3:
  ```bash
  local domain="$1"
  local rule="||${domain}^"
  local current_rules
  current_rules=$(ag_api GET /filtering/status | python3 -c "
  import sys, json
  d = json.load(sys.stdin)
  for r in d.get('user_rules', []):
      print(r)
  ")
  # Check rule exists
  echo "$current_rules" | grep -qF "$rule" || die "No block rule found for: $domain"
  confirm_action "Remove block rule for '$domain'?"
  local new_rules
  new_rules=$(echo "$current_rules" | RULE="$rule" python3 -c "
  import sys, os
  rule = os.environ['RULE']
  rules = [r for r in sys.stdin.read().strip().split('\n') if r != rule]
  import json; print(json.dumps(rules))
  ")
  ag_api POST /filtering/set_rules -d "{\"rules\":$new_rules}"
  ok "  Removed block rule: $rule"
  ```

**cmd_clients()** (per D-08, AG-07):
- Call `ag_api GET /clients`
- Parse JSON with python3: response has `clients` array of persistent clients
- Print table with columns: Name, IDs (comma-joined list of IPs/MACs from `ids` array), Filtering (use_global_settings or custom)
- Use `header "CLIENTS"`
- Handle empty: "  No persistent clients configured."

```python
import sys, json
d = json.load(sys.stdin)
clients = d.get('clients', [])
if not clients:
    print('  No persistent clients configured.')
    sys.exit()
print(f'  {"Name":<25} {"IDs":<35} {"Filtering"}')
print(f'  {"─"*24} {"─"*34} {"─"*15}')
for c in clients:
    name = c.get('name', '?')
    ids = ', '.join(c.get('ids', []))
    if len(ids) > 34:
        ids = ids[:31] + '...'
    filt = 'Global' if c.get('use_global_settings', True) else 'Custom'
    blocked = c.get('blocked_services', [])
    if blocked:
        filt += f' (+{len(blocked)} blocked)'
    print(f'  {name:<25} {ids:<35} {filt}')
print(f'\n  Total: {len(clients)} clients')
```

**cmd_client()** (per D-09, AG-08):
- Usage: `adguard client <name>`
- Validate 1 arg or die: `die "Usage: adguard client <name>"`
- Call `ag_api GET /clients`, filter by name (case-insensitive substring match)
- Show detailed client config: name, IDs, tags, use_global_settings, filtering_enabled, safe_search, blocked_services list, upstreams
- Use `header "CLIENT: <name>"`

```python
import sys, json, os
name = os.environ['CLIENT_NAME'].lower()
d = json.load(sys.stdin)
match = None
for c in d.get('clients', []):
    if name in c.get('name', '').lower():
        match = c; break
if not match:
    print(f'  No client matching: {os.environ["CLIENT_NAME"]}')
    sys.exit(1)
c = match
print(f'  Name:             {c.get("name", "?")}')
print(f'  IDs:              {", ".join(c.get("ids", []))}')
tags = c.get('tags', [])
print(f'  Tags:             {", ".join(tags) if tags else "none"}')
print(f'  Global Settings:  {"Yes" if c.get("use_global_settings") else "No"}')
print(f'  Filtering:        {"Enabled" if c.get("filtering_enabled") else "Disabled"}')
ss = c.get('safe_search', {})
if isinstance(ss, dict):
    print(f'  Safe Search:      {"Enabled" if ss.get("enabled") else "Disabled"}')
else:
    print(f'  Safe Search:      {"Enabled" if ss else "Disabled"}')
blocked = c.get('blocked_services', [])
print(f'  Blocked Services: {", ".join(blocked) if blocked else "none"}')
upstreams = c.get('upstreams', [])
print(f'  Upstreams:        {", ".join(upstreams) if upstreams else "default"}')
```
