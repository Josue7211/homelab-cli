**cmd_rewrites()** (per D-03, AG-03):
- Call `ag_api GET /rewrite/list`
- Parse JSON array with inline python3
- Print table with columns: Domain, Answer
- Use `header "DNS REWRITES"`
- Handle empty: "  No DNS rewrites configured."

```python
import sys, json
data = json.load(sys.stdin)
if not data:
    print('  No DNS rewrites configured.')
    sys.exit()
print(f'  {"Domain":<40} {"Answer"}')
print(f'  {"─"*39} {"─"*30}')
for r in data:
    domain = r.get('domain', '?')
    answer = r.get('answer', '?')
    print(f'  {domain:<40} {answer}')
print(f'\n  Total: {len(data)} rewrites')
```

**cmd_rewrite_add()** (per D-04, AG-04):
- Usage: `adguard rewrite-add <domain> <answer>`
- Validate 2 args or die with usage: `die "Usage: adguard rewrite-add <domain> <answer>"`
- Call `ag_api POST /rewrite/add -d "{\"domain\":\"$1\",\"answer\":\"$2\"}"`
- On success: `ok "  Added rewrite: $1 -> $2"`

**cmd_rewrite_rm()** (per D-05, AG-05):
- Usage: `adguard rewrite-rm <domain>`
- Validate 1 arg or die with usage: `die "Usage: adguard rewrite-rm <domain>"`
- First GET the rewrite list to find the matching entry's answer:
  ```bash
  local domain="$1"
  local answer
  answer=$(ag_api GET /rewrite/list | DOMAIN="$domain" python3 -c "
  import sys, json, os
  domain = os.environ['DOMAIN']
  for r in json.load(sys.stdin):
      if r.get('domain') == domain:
          print(r.get('answer', '')); break
  else:
      print('NOT_FOUND')
  ")
  [[ "$answer" == "NOT_FOUND" ]] && die "No rewrite found for domain: $domain"
  ```
- Call `confirm_action "Remove DNS rewrite '$domain' -> '$answer'?"`
- Call `ag_api POST /rewrite/delete -d "{\"domain\":\"$domain\",\"answer\":\"$answer\"}"`
- On success: `ok "  Removed rewrite: $domain -> $answer"`

Wire all 3 into main() case statement after dhcp entries:

- `rewrite-add)             cmd_rewrite_add "$@" ;;`
