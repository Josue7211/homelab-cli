   - Check KOEL_EMAIL and KOEL_PASS are set, die if not with message "Set KOEL_EMAIL and KOEL_PASS in ~/.config/homelab-cli/config"
   - POST to "${KOEL_URL}/api/me" with JSON body {"email":"${KOEL_EMAIL}","password":"${KOEL_PASS}"} using python3 + os.environ for safe JSON construction: python3 -c "import json,os; print(json.dumps({'email':os.environ['KOEL_EMAIL'],'password':os.environ['KOEL_PASS']}))"

   - mkdir -p "$(dirname "$KOEL_TOKEN_FILE")" and write token to $KOEL_TOKEN_FILE
   - echo the token

3. Add koel_token() function:
   - If KOEL_TOKEN_FILE exists and is less than 24 hours old (find "$KOEL_TOKEN_FILE" -mmin -1440 2>/dev/null), cat and return it
   - Otherwise call koel_login

4. Add koel_api() function (per D-00, modeled on jellyfin_api):
   - Takes method as $1, path as $2, data as optional $3
   - Gets token via koel_token
   - Constructs Authorization: Bearer ${token} header
   - Dispatches GET/POST/DELETE via curl -sf with appropriate headers
   - URL: "${KOEL_URL}${path}"

5. Add cmd_search (per D-01, KOEL-01):
   - Usage guard: die "Usage: koel search <query>" if no $1
   - Capture query="$*" (allow multi-word)
   - URL-encode query using python3: python3 -c "import urllib.parse; print(urllib.parse.quote('$query'))" -- but safer: pass via env var QUERY="$query" python3 -c "import urllib.parse,os; print(urllib.parse.quote(os.environ['QUERY']))"
   - GET /api/search?q=${encoded}
   - Parse with python3, display results grouped into 3 sections:
     SONGS header, then each song: title | artist | album | duration (format seconds to M:SS)
     ALBUMS header, then each album: name | artist | year
     ARTISTS header, then each artist: name
   - If no results in any category, print "(none)"

6. Add cmd_playlists (per D-02, KOEL-02):
   - GET /api/playlist
   - Parse JSON array with python3
   - Display table with columns: ID | Name | Songs (count of songs array length) | Owner (if present, else "-")
   - Use header "PLAYLISTS" and format_table via pipe

7. Add case statement entries in main():
   - search) cmd_search "$@" ;;
   - playlists) cmd_playlists ;;
