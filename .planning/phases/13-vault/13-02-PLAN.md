**cmd_folders()** (per D-04, BW-04):
- Usage: `vault folders` (no args)
- Step 1: `check_unlocked`
- Step 2: `bw list folders 2>/dev/null` to get all folders
- Step 3: Display with python3 as a formatted list:
  - Print header "VAULT FOLDERS"
  - Print "(no folder)" entry first as a reminder for unorganized items
  - For each folder: print ID and Name
  - Show total count
- Die if fetch fails: `die "Failed to list folders"`

```bash
cmd_folders() {
    check_unlocked

    local folders_json
    folders_json=$(bw list folders 2>/dev/null) || die "Failed to list folders"

    header "VAULT FOLDERS"

    echo "$folders_json" | python3 -c "
import sys, json
folders = json.load(sys.stdin)
print(f'  Total: {len(folders)} folders\n')
print(f'  {\"(no folder)\":<40} Items without a folder')
print()
print(f'  {\"ID\":<40} {\"Name\"}')
print(f'  {\"─\"*38}   {\"─\"*30}')
for f in sorted(folders, key=lambda x: x.get('name', '')):
    fid = f.get('id', '?')
    name = f.get('name', '?')
    print(f'  {fid:<40} {name}')
"
}
```

**cmd_generate()** (per D-05, BW-05):
- Usage: `vault generate [--length N] [--no-symbols] [--numbers-only]`
- Default length: 20
- Parse optional flags: --length N, --no-symbols, --numbers-only
- Build bw generate command with appropriate flags:
  - Always: `bw generate --length $length`
  - Default (no flags): includes uppercase, lowercase, number, special (`-ulns`)
  - `--no-symbols`: uppercase, lowercase, number only (`-uln`)
  - `--numbers-only`: number only (`-n`)
- Print the generated password directly (bw generate outputs to stdout)

```bash
cmd_generate() {
    local length=20
    local charset="-ulns"  # uppercase, lowercase, number, special

    while [[ $# -gt 0 ]]; do
        case "$1" in
            --length)       length="${2:-20}"; shift 2 ;;
            --no-symbols)   charset="-uln"; shift ;;
            --numbers-only) charset="-n"; shift ;;
            *)              die "Unknown option: $1" ;;
        esac
    done

    local pw
    pw=$(bw generate --length "$length" $charset 2>/dev/null) || die "Failed to generate password"
    echo "$pw"
}
```

**cmd_totp()** (per D-06, BW-06):
- Usage: `vault totp <id|name>`
- Validate arg present or die: `die "Usage: vault totp <id|name>"`
- Step 1: `check_unlocked`
- Step 2: `bw get totp "$1" 2>/dev/null` to get the current TOTP code
- Step 3: Display the code
- Step 4: Show remaining seconds until code rotates (TOTP standard = 30-second period, remaining = 30 - (epoch % 30))
- Die if item has no TOTP: `die "No TOTP configured for '$1'"`

```bash
cmd_totp() {
    local identifier="${1:-}"
    [[ -z "$identifier" ]] && die "Usage: vault totp <id|name>"

    check_unlocked

    local code
    code=$(bw get totp "$identifier" 2>/dev/null) || die "No TOTP configured for '$identifier'"

    [[ -z "$code" ]] && die "No TOTP configured for '$identifier'"

    local remaining
    remaining=$(python3 -c "import time; print(30 - int(time.time()) % 30)")

    echo -e "  Code:      ${BOLD}${code}${RESET}"
    echo -e "  Expires:   ${remaining}s"
}
```
