- Usage: `sab delete <nzo_id> [--del-files]`
- Validate at least 1 arg or die: `die "Usage: sab delete <nzo_id> [--del-files]"`
- Call `confirm_action "Delete item '$nzo_id' from queue?"`
- If `--del-files` flag present, use `name=delete&value=$nzo_id&del_files=1`; otherwise `name=delete&value=$nzo_id`
- Call `sab_api "queue" "name=delete" "value=$nzo_id"` (plus `"del_files=1"` if flag set)
- Parse result: check for `{"status": true}` in response
- On success: `ok "  Deleted from queue: $nzo_id"`
- On failure: `die "Failed to delete item: $nzo_id"`

```bash
cmd_delete() {
    local nzo_id="${1:-}"
    [[ -z "$nzo_id" ]] && die "Usage: sab delete <nzo_id> [--del-files]"
    local del_files=0
    [[ "${2:-}" == "--del-files" ]] && del_files=1
    confirm_action "Delete item '$nzo_id' from queue?"
    local result
    if [[ "$del_files" -eq 1 ]]; then
        result=$(sab_api "queue" "name=delete" "value=$nzo_id" "del_files=1") || die "Failed to delete item: $nzo_id"
    else
        result=$(sab_api "queue" "name=delete" "value=$nzo_id") || die "Failed to delete item: $nzo_id"
    fi
    echo "$result" | python3 -c "
import sys, json
d = json.load(sys.stdin)
if d.get('status'):
    pass
else:
    print(f'  Error: {d.get(\"error\", \"unknown\")}')
    sys.exit(1)
" || die "Failed to delete item: $nzo_id"
    ok "  Deleted from queue: $nzo_id"
}
```

**cmd_priority()** (per D-05, SAB-05):
- Usage: `sab priority <nzo_id> <level>`
- Validate 2 args or die: `die "Usage: sab priority <nzo_id> <stop|low|normal|high|force>"`
- Map named levels to numeric: stop=-100, low=-1, normal=0, high=1, force=2
- If numeric value passed directly (-100,-1,0,1,2), use as-is
- Call `sab_api "queue" "name=priority" "value=$nzo_id" "value2=$numeric_level"`
- On success: `ok "  Priority set to $level for $nzo_id"`

```bash
cmd_priority() {
    local nzo_id="${1:-}"
    local level="${2:-}"
    [[ -z "$nzo_id" || -z "$level" ]] && die "Usage: sab priority <nzo_id> <stop|low|normal|high|force>"
    local numeric
    case "$level" in
        stop)    numeric="-100" ;;
        low)     numeric="-1" ;;
        normal)  numeric="0" ;;
        high)    numeric="1" ;;
        force)   numeric="2" ;;
        -100|-1|0|1|2) numeric="$level" ;;
        *)       die "Invalid priority: $level (use: stop, low, normal, high, force)" ;;
    esac
    sab_api "queue" "name=priority" "value=$nzo_id" "value2=$numeric" >/dev/null || die "Failed to set priority"
    ok "  Priority set to $level for $nzo_id"
}
```
