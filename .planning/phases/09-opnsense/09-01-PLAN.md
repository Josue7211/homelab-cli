Add 3 functions before cmd_help() in bin/opnsense:

**cmd_rule_add()** (per D-01, OPN-01):
- Usage: `opnsense rule-add <description> <action> <interface> <source> <destination>`
- Validate 5 args or die: `die "Usage: opnsense rule-add <description> <action> <interface> <source> <destination>"`
- Build JSON payload with python3 for proper escaping:
  ```bash
  local desc="$1" action="$2" iface="$3" src="$4" dst="$5"
  local payload
  payload=$(python3 -c "
  import json, sys
  print(json.dumps({'rule': {
      'description': '$desc',
      'action': '$action',
      'interface': '$iface',
      'source_net': '$src',
      'destination_net': '$dst',
      'enabled': '1'
  }}))
  ")
  ```
  NOTE: Use environment variables (DESC="$desc" DST="$dst" etc. + os.environ) instead of shell interpolation in the python3 string to avoid injection. Pattern:
  ```bash
  payload=$(DESC="$desc" ACTION="$action" IFACE="$iface" SRC="$src" DST="$dst" python3 -c "
  import json, os
  print(json.dumps({'rule': {
      'description': os.environ['DESC'],
      'action': os.environ['ACTION'],
      'interface': os.environ['IFACE'],
      'source_net': os.environ['SRC'],
      'destination_net': os.environ['DST'],
      'enabled': '1'
  }}))
  ")
  ```
- Call `opn_post "/firewall/filter/addRule" "$payload"`
- Parse response with python3 to extract uuid: `result.get('uuid', '')`
- If uuid present: `ok "  Rule added (uuid: $uuid)"` then `warn "  Run 'opnsense apply' to activate changes"`
- If no uuid: `die "Failed to add rule"`
